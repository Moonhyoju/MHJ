TCP/IP 소켓 통신을 이용한 C채팅 프로그램
-----------------------------------------
소켓 통신을 이용한 C채팅 프로그램입니다.

**Client**
-----------------------------------------
1. IP와 Port 번호 설정 후 서버로 접속
2. 서버로부터 받은 메시지 출력
3. 서버와 연결 상태 확인

**Server**
-----------------------------------------
1. Port 번호 설정 후 Client가 접속할 수 있도록 Listen 상태로 대기
2. 다중 Client 접속 허용
3. Client로 부터 받은 메시지를 해당 Client에 Echo
4. 서버에 연결된 모든 Client에 메시지 송신


**기술 스택**
+ TCP/IP
+ Socket 통신
+ C
+ MFC
